Генерация и анализ проводок по выдаче кредитов (Тестовое задание)

Условие задачи
Написать скрипт, генерирующий 1000 случайных проводок по выдаче кредита физическим лицам. На базе полученных проводок
- выбрать 10 самых крупных выдач.
- найти дни, в которые было выдано больше всего кредитов

Атрибуты каждой проводки:
Счет по дебету (debit_account)
Счет по кредиту (credit_account)
Сумма (от 100 до 500 тыс. руб, кратно 1000)
Дата (только в августе 2025, без воскресений)

SQL-анализ

10 самых крупных выдач:

SELECT * FROM credit_postings_august_2025
ORDER BY amount_thousands DESC 
LIMIT 10;

8 дат с наибольшим количеством выдач

SELECT date, COUNT(*) AS num_credits
FROM clean_postings
GROUP BY date
ORDER BY num_credits DESC
LIMIT 8;
